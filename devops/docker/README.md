* 하드웨어 가상화 없는 격리된 환경에서 실행되는 프로세스


# Docker Command

* Docker version check
  ```cmd
  docker -v
  ```
* Docker Image Download
  ```cmd
  docker pull {image-name}:{tag}
  ```
* Docker Image View
  ```cmd
  docker images
  ```
* Create Docker Container
  ```cmd
  docker create {option} {image-name}:{tag}
  ```
